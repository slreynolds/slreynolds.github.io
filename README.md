## Steven Lamarr Reynolds

I'm a research associate in the Information Visualization and Visual Analytics department of Fraunhofer IGD, Germany. My interests are in interactive data visualizations for privacy and security domains with a focus on human-centered design.


[Twitter](https://twitter.com/_slreynolds), [GitHub](https://github.com/slreynolds/), [LinkedIN](https://www.linkedin.com/in/steven-lamarr-reynolds/), [Google Scholar](https://scholar.google.com/citations?user=OEhfmGgAAAAJ), [Orcid](https://orcid.org/0000-0002-0564-2788)


## Featured Projects

* [TransparencyVis](https://transparency-vis.vx.igd.fraunhofer.de/) (2022) - visualization of personal data exports, TypeScript & React & d3
* VUSCVIS (2021) - visualization of software vulnerability for domain experts, TypeScript & React & d3
* [toyblocks][4] (2015) - e-learning platform with games for architecture students, js & express
* Visual Analytics of Topic Dynamics (bachelor's thesis 2019) - visualization of text topic modeling data over sliding windows, JS & React & d3

Other projects:

* [generateTypes][0] (2019) - small tool to generate Typescript Types from JSON files
* buk
* snake
